Para el desarrollo del sistema de gestión de turnos médicos se utilizará la metodología ágil Scrum.  
Se eligió esta metodología porque permite organizar el trabajo en tareas pequeñas y manejables, facilitando el seguimiento del progreso.  
Además, Scrum mejora la comunicación entre los integrantes del equipo y permite adaptarse a cambios durante el desarrollo.  
El uso de un tablero Kanban permite visualizar el estado de las tareas y optimizar la organización del equipo.  
Por estas razones, Scrum resulta adecuada para el desarrollo de este proyecto.
